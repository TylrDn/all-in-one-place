# Content Creation

MCP/AI agent workflows for creating various types of content.

## Subdirectories

- **copywriting**: Workflows for generating marketing copy and content
- **image-generation**: Workflows for creating and editing images with AI
- **video-editing**: Workflows for automating video editing tasks
- **social-media**: Workflows for creating and scheduling social media content

## Use Cases

- Generate marketing copy
- Create AI-generated images
- Automate video editing
- Schedule social media posts
