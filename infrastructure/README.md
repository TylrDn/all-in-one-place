# Infrastructure

MCP/AI agent workflows for cloud and infrastructure management.

## Subdirectories

- **cloud-management**: Workflows for managing cloud resources
- **configuration**: Workflows for configuration management
- **container-orchestration**: Workflows for managing containers and Kubernetes
- **serverless**: Workflows for serverless application deployment

## Use Cases

- Manage cloud infrastructure
- Automate configuration management
- Orchestrate containers
- Deploy serverless applications
