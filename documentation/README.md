# Documentation

MCP/AI agent workflows for creating and maintaining documentation.

## Subdirectories

- **api-docs**: Workflows for generating API documentation from code
- **technical-writing**: Workflows for creating technical documentation and guides
- **documentation-generation**: Automated documentation generation workflows
- **translation**: Workflows for translating documentation to multiple languages

## Use Cases

- Generate API documentation automatically
- Create user guides and tutorials
- Keep documentation in sync with code
- Translate documentation for international audiences
