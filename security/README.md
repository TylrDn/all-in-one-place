# Security

MCP/AI agent workflows for security scanning, compliance, and auditing.

## Subdirectories

- **vulnerability-scanning**: Workflows for detecting security vulnerabilities
- **security-audit**: Workflows for security assessment and penetration testing
- **compliance**: Workflows for ensuring regulatory compliance (GDPR, HIPAA, etc.)
- **secrets-management**: Workflows for managing secrets and credentials securely

## Use Cases

- Scan code for vulnerabilities
- Perform security audits
- Ensure compliance with regulations
- Manage secrets and API keys
