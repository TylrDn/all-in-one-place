# API Integration

MCP/AI agent workflows for integrating with external APIs and services.

## Subdirectories

- **rest-api**: Workflows for REST API integration and testing
- **graphql**: Workflows for GraphQL API integration and query building
- **webhooks**: Workflows for handling and managing webhooks
- **third-party-services**: Workflows for integrating with third-party services

## Use Cases

- Integrate with REST APIs
- Build GraphQL queries and mutations
- Handle webhook events
- Connect to third-party services
