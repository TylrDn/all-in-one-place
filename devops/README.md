# DevOps

MCP/AI agent workflows for CI/CD, deployment, and operations.

## Subdirectories

- **ci-cd**: Continuous Integration and Continuous Deployment workflows
- **deployment**: Workflows for deploying applications to various environments
- **monitoring**: Workflows for system monitoring and alerting
- **infrastructure-as-code**: Workflows for managing infrastructure with code

## Use Cases

- Automate build and deployment pipelines
- Monitor application health
- Manage infrastructure with IaC tools
- Set up automated alerting
