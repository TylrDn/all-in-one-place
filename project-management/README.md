# Project Management

MCP/AI agent workflows for project planning and team coordination.

## Subdirectories

- **task-automation**: Workflows for automating project tasks
- **planning**: Workflows for project planning and estimation
- **reporting**: Workflows for generating project reports
- **resource-management**: Workflows for managing team resources

## Use Cases

- Automate repetitive project tasks
- Create project plans and estimates
- Generate status reports
- Manage team resources and capacity
