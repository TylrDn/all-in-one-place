# ML/AI Operations

MCP/AI agent workflows for machine learning and AI model operations.

## Subdirectories

- **model-training**: Workflows for training machine learning models
- **model-deployment**: Workflows for deploying models to production
- **mlops**: Workflows for ML operations and lifecycle management
- **feature-engineering**: Workflows for creating and selecting features

## Use Cases

- Train machine learning models
- Deploy models to production
- Manage ML pipelines
- Engineer features for models
