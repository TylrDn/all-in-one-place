# Localization

MCP/AI agent workflows for internationalization and localization.

## Subdirectories

- **translation-automation**: Workflows for automated translation
- **i18n**: Internationalization workflows and best practices
- **l10n**: Localization workflows for specific regions
- **locale-management**: Workflows for managing locale files and resources

## Use Cases

- Automate translation processes
- Implement internationalization
- Localize applications for specific regions
- Manage translation resources
