# Research

MCP/AI agent workflows for information gathering and analysis.

## Subdirectories

- **information-gathering**: Workflows for collecting information from various sources
- **summarization**: Workflows for summarizing documents and reports
- **competitive-analysis**: Workflows for analyzing competitors and market trends
- **literature-review**: Workflows for reviewing academic papers and technical literature

## Use Cases

- Gather information from multiple sources
- Summarize long documents
- Analyze market competitors
- Review academic literature
