# Testing

MCP/AI agent workflows for test automation and quality assurance.

## Subdirectories

- **unit-testing**: Workflows for generating and running unit tests
- **integration-testing**: Workflows for testing component interactions
- **e2e-testing**: End-to-end testing workflows for full application testing
- **performance-testing**: Workflows for load testing and performance analysis

## Use Cases

- Generate test cases automatically
- Run comprehensive test suites
- Perform load and stress testing
- Analyze test coverage
