# Communication

MCP/AI agent workflows for automating communication tasks.

## Subdirectories

- **email-automation**: Workflows for automated email handling and responses
- **chat-integration**: Workflows for integrating with chat platforms
- **notifications**: Workflows for sending notifications across various channels
- **slack-bots**: Workflows for creating Slack bots and automations

## Use Cases

- Automate email responses
- Integrate with team chat platforms
- Send notifications to team members
- Build custom Slack bots
