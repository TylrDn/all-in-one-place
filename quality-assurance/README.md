# Quality Assurance

MCP/AI agent workflows for code quality and static analysis.

## Subdirectories

- **code-quality**: Workflows for analyzing and improving code quality
- **static-analysis**: Workflows for static code analysis
- **code-coverage**: Workflows for measuring and improving test coverage
- **linting**: Workflows for automated linting and style checking

## Use Cases

- Analyze code quality metrics
- Run static analysis tools
- Measure test coverage
- Enforce coding standards
