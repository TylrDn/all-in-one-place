# Analytics

MCP/AI agent workflows for data analytics and business intelligence.

## Subdirectories

- **business-intelligence**: Workflows for BI reporting and insights
- **metrics-dashboards**: Workflows for creating metrics dashboards
- **data-visualization**: Workflows for creating data visualizations
- **reporting**: Workflows for automated analytics reporting

## Use Cases

- Generate business intelligence reports
- Create real-time dashboards
- Visualize complex data
- Automate analytics reporting
