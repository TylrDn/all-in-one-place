# Accessibility

MCP/AI agent workflows for web accessibility and inclusive design.

## Subdirectories

- **wcag-compliance**: Workflows for WCAG compliance testing
- **screen-reader-testing**: Workflows for testing with screen readers
- **accessibility-audit**: Workflows for accessibility auditing
- **inclusive-design**: Workflows for implementing inclusive design

## Use Cases

- Test WCAG compliance
- Verify screen reader compatibility
- Conduct accessibility audits
- Implement inclusive design principles
