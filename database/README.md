# Database

MCP/AI agent workflows for database management and optimization.

## Subdirectories

- **query-optimization**: Workflows for optimizing database queries
- **schema-design**: Workflows for designing database schemas
- **migration**: Workflows for database migrations and version control
- **backup-recovery**: Workflows for database backup and recovery

## Use Cases

- Optimize slow queries
- Design efficient database schemas
- Manage database migrations
- Automate backup and recovery
